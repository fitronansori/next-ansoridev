# Membuat Halaman HTML Pertama

Mari kita buat halaman HTML pertama! Ini akan sangat mudah.

## Langkah 1: Buka Text Editor

Buka text editor favorit Anda (Notepad, Visual Studio Code, atau lainnya).

## Langkah 2: Tulis Kode HTML

Ketik kode berikut persis seperti ini:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama Saya</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>Ini adalah halaman HTML pertama saya.</p>
    <p>Saya sedang belajar HTML dan sangat senang!</p>
  </body>
</html>
```

## Langkah 3: Simpan File

4. Simpan di Desktop atau folder yang mudah ditemukan

## Langkah 4: Buka di Browser

1. Buka File Explorer (Windows) atau Finder (Mac)
2. Cari file `hello.html` yang sudah Anda simpan
3. Double-click file tersebut
4. File akan terbuka di browser

## Apa yang Anda Lihat?

Anda akan melihat:

- Judul besar "Hello World!"
- Dua paragraf text

## Penjelasan Kode

- `<!DOCTYPE html>` - Memberitahu browser ini HTML5
- `<html>` - Membuka dokumen HTML
- `<head>` - Bagian informasi halaman
- `<title>` - Judul di tab browser
- `<body>` - Bagian yang terlihat di halaman
- `<h1>` - Judul besar
- `<p>` - Paragraf text
