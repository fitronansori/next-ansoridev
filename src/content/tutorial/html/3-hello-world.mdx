# Hello World HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web. Dalam tutorial ini, kita akan membuat halaman web sederhana yang menampilkan pesan "Hello World".

## Langkah 1: Buat File HTML

Buat file baru dengan nama `index.html` di dalam folder proyek Anda. Anda dapat menggunakan teks editor seperti Visual Studio Code, Sublime Text, atau Notepad++ untuk membuat file ini.

![Gambar](https://8j9cli1gc6.ufs.sh/f/RJIZsHcy1birJQM6ouYqvkPSo57cfmXntwyugp0KdNUD98aH)

## Langkah 2: Tulis Kode HTML

Salin dan tempel kode berikut ke dalam file `index.html` yang baru Anda buat:

```html
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>Selamat datang di tutorial HTML.</p>
  </body>
</html>
```

## Langkah 3: Simpan File

Simpan file `index.html` setelah menambahkan kode di atas.

![Gambar](https://8j9cli1gc6.ufs.sh/f/RJIZsHcy1birp6XCxCxWC1cANfMI0F73wldqRhDJSbQKYkZ6)

## Langkah 4: Buka File di Browser

Buka file `index.html` di browser web Anda. Anda dapat melakukannya dengan mengklik dua kali file tersebut atau dengan membuka browser dan menyeret file ke jendela browser.

## Langkah 5: Lihat Hasilnya
