# Apa itu HTML?

HTML adalah singkatan dari **HyperText Markup Language**. HTML adalah bahasa yang digunakan untuk membuat halaman web.

## Mengapa Belajar HTML?

HTML adalah dasar dari semua website. Semua website di internet menggunakan HTML. Jika Anda ingin membuat website, Anda harus belajar HTML terlebih dahulu.

## HTML itu Mudah!

HTML sangat mudah dipelajari. HTML menggunakan **tag** untuk memberitahu browser bagaimana menampilkan text.

Contoh tag HTML:

- `<h1>` untuk membuat judul besar
- `<p>` untuk membuat paragraf
- `<strong>` untuk membuat text tebal

## Struktur Dasar HTML

Setiap halaman HTML memiliki struktur dasar seperti ini:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Selamat Datang!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>
```

## Penjelasan Sederhana

- `<!DOCTYPE html>` - Memberitahu browser ini adalah HTML5
- `<html>` - Tag pembuka untuk semua HTML
- `<head>` - Berisi informasi tentang halaman
- `<title>` - Judul yang muncul di tab browser
- `<body>` - Berisi semua yang terlihat di halaman
- `<h1>` - Judul besar
- `<p>` - Paragraf text

## Yang Akan Kita Pelajari

Dalam tutorial ini, kita akan belajar:

1. **Dasar-dasar HTML** - Apa itu HTML dan mengapa penting
2. **Alat yang diperlukan** - Text editor dan browser
3. **Membuat halaman pertama** - Hello World dalam HTML
4. **Elemen-elemen dasar** - Tag, atribut, dan struktur
5. **Text dan formatting** - Heading, paragraf, dan text formatting
6. **Link dan gambar** - Cara membuat link dan menampilkan gambar
7. **List dan tabel** - Membuat daftar dan tabel data
8. **Form** - Membuat form input sederhana
9. **Semantic HTML** - Menggunakan tag yang bermakna

## Mengapa HTML Penting?

HTML adalah **fondasi** dari semua website. Tanpa HTML:

- Tidak ada text di website
- Tidak ada struktur halaman
- Tidak ada link antar halaman
- Tidak ada gambar yang bisa ditampilkan

Dengan mempelajari HTML, Anda akan memahami bagaimana website bekerja dari dasar.

## Tips Belajar HTML

1. **Praktik langsung** - Tulis kode, jangan hanya baca
2. **Mulai sederhana** - Jangan terburu-buru ke hal rumit
3. **Eksperimen** - Coba ubah-ubah kode dan lihat hasilnya
4. **Sabar** - Setiap programmer pernah menjadi pemula
