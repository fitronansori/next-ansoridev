# Tag-Tag Dasar HTML

Mari belajar tag-tag HTML yang paling sering digunakan! Semuanya sangat mudah.

## Tag Heading (Judul)

HTML punya 6 ukuran judul, dari besar ke kecil:

```html
<h1>Judul Paling Besar</h1>
<h2>Judul Besar</h2>
<h3>Judul Sedang</h3>
<h4>Judul Kecil</h4>
<h5>Judul Lebih Kecil</h5>
<h6>Judul Paling Kecil</h6>
```

**Tips:** Gunakan `<h1>` untuk judul utama halaman.

## Tag Paragraf

Untuk membuat paragraf text:

```html
<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>
```

## Tag Pemformatan Text

Untuk membuat text khusus:

```html
<strong>Text tebal (bold)</strong> <em>Text miring (italic)</em>
```

Contoh dalam kalimat:

```html
<p>
  HTML adalah bahasa <strong>markup</strong> yang <em>mudah</em> dipelajari.
</p>
```

## Tag Garis Baru

Untuk membuat baris baru:

```html
<p>Baris pertama<br />Baris kedua</p>
```

`<br>` tidak perlu tag penutup.

## Tag Garis Horizontal

Untuk membuat garis pemisah:

```html
<p>Bagian pertama</p>
<hr />
<p>Bagian kedua</p>
```

`<hr>` juga tidak perlu tag penutup.

## Contoh Lengkap

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Tag-Tag Dasar</title>
  </head>
  <body>
    <h1>Selamat Datang di Website Saya</h1>

    <p>Halo! Nama saya <strong>Budi</strong>.</p>
    <p>Saya <em>sangat senang</em> belajar HTML.</p>

    <hr />

    <h2>Tentang Saya</h2>
    <p>
      Saya tinggal di Jakarta.<br />
      Saya suka coding dan membaca buku.
    </p>
  </body>
</html>
```

## Latihan

Coba buat halaman dengan:

1. Judul utama (`<h1>`)
2. Subjudul (`<h2>`)
3. 2 paragraf dengan text tebal dan miring
4. Garis pemisah (`<hr>`)
5. Paragraf dengan baris baru (`<br>`)

Selamat mencoba!
