# List dan Tabel

Mari belajar cara membuat daftar (list) dan tabel di HTML. Kedua elemen ini sangat berguna untuk mengorganisir informasi!

## List Tidak Berurut (Unordered List)

Untuk membuat daftar dengan bullet points:

```html
<ul>
  <li>Apel</li>
  <li>Jeruk</li>
  <li>Mangga</li>
</ul>
```

**Hasilnya:**

- Apel
- Jeruk
- Mangga

## List Berurut (Ordered List)

Untuk membuat daftar dengan nomor:

```html
<ol>
  <li>Bangun tidur</li>
  <li>Mandi</li>
  <li>Sarapan</li>
  <li>Berangkat kerja</li>
</ol>
```

**Hasilnya:**

1. Bangun tidur
2. Mandi
3. Sarapan
4. Berangkat kerja

## List Bertingkat (Nested List)

List di dalam list:

```html
<ul>
  <li>
    Buah-buahan
    <ul>
      <li>Apel</li>
      <li>Jeruk</li>
    </ul>
  </li>
  <li>
    Sayuran
    <ul>
      <li>Bayam</li>
      <li>Kangkung</li>
    </ul>
  </li>
</ul>
```

## Tabel Dasar

Untuk membuat tabel data:

```html
<table>
  <tr>
    <th>Nama</th>
    <th>Umur</th>
    <th>Kota</th>
  </tr>
  <tr>
    <td>Budi</td>
    <td>25</td>
    <td>Jakarta</td>
  </tr>
  <tr>
    <td>Sari</td>
    <td>23</td>
    <td>Bandung</td>
  </tr>
</table>
```

**Penjelasan Tag Tabel:**

- `<table>` = tag pembuka tabel
- `<tr>` = table row (baris)
- `<th>` = table header (judul kolom)
- `<td>` = table data (data/isi)

## Tabel dengan Border

Untuk membuat garis tabel:

```html
<table border="1">
  <tr>
    <th>Produk</th>
    <th>Harga</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>Rp 5.000.000</td>
  </tr>
  <tr>
    <td>Mouse</td>
    <td>Rp 100.000</td>
  </tr>
</table>
```

## Tips List

1. **Gunakan `<ul>`** untuk daftar yang tidak perlu urutan
2. **Gunakan `<ol>`** untuk langkah-langkah atau ranking
3. **List bertingkat** bagus untuk kategori dan sub-kategori
4. **Setiap item** harus dalam tag `<li>`

## Tips Tabel

1. **Selalu gunakan `<th>`** untuk header/judul kolom
2. **Gunakan `border="1"`** untuk melihat garis tabel
3. **Pastikan jumlah `<td>`** sama di setiap baris
4. **Tabel bagus** untuk data terstruktur seperti harga, jadwal, dll

## Atribut Tabel Lainnya

```html
<!-- Tabel dengan caption -->
<table border="1">
  <caption>
    Daftar Siswa Kelas 12A
  </caption>
  <tr>
    <th>No</th>
    <th>Nama</th>
    <th>Nilai</th>
  </tr>
  <tr>
    <td>1</td>
    <td>Ahmad</td>
    <td>85</td>
  </tr>
</table>
```
