# Alat untuk Belajar HTML

Untuk belajar HTML, Anda hanya memerlukan 2 hal sederhana:

1. **Text Editor** - untuk menulis kode HTML
2. **Web Browser** - untuk melihat hasil kode HTML

## Text Editor

Anda bisa menggunakan text editor apapun. Berikut beberapa pilihan:

### Notepad (Windows)

- Sudah ada di Windows
- Gratis dan mudah
- Cocok untuk pemula

### Visual Studio Code (Disarankan)

- Gratis dan sangat bagus
- Ada warna untuk kode (syntax highlighting)
- Bisa download di: https://code.visualstudio.com

### TextEdit (Mac)

- Sudah ada di Mac
- Pastikan pilih "Plain Text" mode

## Web Browser

Browser adalah program untuk melihat website. Semua browser bisa membuka file HTML:

### Google Chrome (Disarankan)

- Paling populer dan cepat
- Tools untuk developer yang bagus
- Download di: https://www.google.com/chrome

### Mozilla Firefox

- Open source dan privacy-focused
- Tools developer yang lengkap
- Download di: https://www.mozilla.org/firefox

### Microsoft Edge

- Sudah ada di Windows 10/11
- Cepat dan modern

## Cara Membuat File HTML

Langkah mudah membuat file HTML:

1. **Buka text editor** (Notepad, VS Code, dll)
2. **Tulis kode HTML**
3. **Save dengan ekstensi .html**
   - Contoh: `halaman-pertama.html`
   - Jangan lupa ekstensi `.html`!
4. **Double click file** untuk membuka di browser

## Tips Penting

### Penamaan File

- Gunakan huruf kecil: `index.html` ✅
- Jangan spasi: `halaman pertama.html` ❌
- Gunakan dash: `halaman-pertama.html` ✅
- Jangan karakter khusus: `halaman#1.html` ❌

### Encoding

Pastikan save file dengan encoding **UTF-8** agar karakter Indonesia muncul dengan benar.

## Siap Mulai?

Dengan text editor dan browser, Anda sudah siap membuat website pertama!

Mari lanjut ke tutorial berikutnya untuk membuat halaman HTML pertama Anda.

Anda bisa menggunakan browser apapun:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

Semua browser modern bisa menampilkan HTML dengan baik.

## Cara Memulai

1. Buka text editor
2. Tulis kode HTML
3. Simpan file dengan ekstensi `.html` (contoh: `halaman.html`)
4. Buka file tersebut dengan browser

## Tips Sederhana

- Buat folder khusus untuk latihan HTML
- Beri nama file yang mudah diingat
- Selalu simpan file dengan ekstensi `.html`
- Buka file di browser untuk melihat hasilnya

Itu saja! Sekarang Anda siap membuat halaman HTML pertama.
