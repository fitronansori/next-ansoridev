# JavaScript Dasar: Memulai Perjalanan Coding

JavaScript adalah bahasa pemrograman yang sangat mudah dipelajari. Mari kita mulai dengan konsep-konsep dasar tanpa perlu ribet dengan HTML atau CSS.

## Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk:

- Membuat logika program
- Mengolah data
- Membuat perhitungan
- Membuat keputusan otomatis

Bayangkan JavaScript seperti **otak** yang bisa berpikir dan membuat keputusan.

## JavaScript Pertama Anda

Mari tulis kode JavaScript pertama di console:

```javascript
console.log("Halo, saya sedang belajar JavaScript!");
```

**Penjelasan:**

- `console.log()` = perintah untuk menampilkan pesan
- Tanda kutip (`"`) = menandai text/string
- Titik koma (`;`) = mengakhiri perintah

## Mencoba Perintah Sederhana

### 1. Menampilkan Pesan

```javascript
console.log("Nama saya Budi");
console.log("Saya berumur 20 tahun");
console.log("Saya suka coding!");
```

### 2. Perhitungan Matematika

```javascript
console.log(5 + 3); // Hasil: 8
console.log(10 - 4); // Hasil: 6
console.log(7 * 2); // Hasil: 14
console.log(15 / 3); // Hasil: 5
```

### 3. Menggabungkan Text

```javascript
console.log("Halo" + " " + "Dunia"); // Hasil: Halo Dunia
console.log("Saya punya " + 5 + " buku"); // Hasil: Saya punya 5 buku
```

## Latihan Praktek

**Coba di console browser:**

1. Tampilkan nama dan hobby Anda
2. Hitung umur Anda 10 tahun lagi
3. Gabungkan kalimat "Saya tinggal di" dengan nama kota Anda

**Contoh jawaban:**

```javascript
console.log("Nama saya Sarah, hobby saya membaca");
console.log("Umur saya sekarang: " + 25);
console.log("Umur 10 tahun lagi: " + (25 + 10));
console.log("Saya tinggal di " + "Jakarta");
```
